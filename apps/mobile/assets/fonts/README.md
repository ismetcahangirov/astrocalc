# AstroSymbols.ttf

A tiny (~6 KB) subset font containing only the zodiac-sign and celestial-body
glyphs the natal-chart wheel renders (Unicode Miscellaneous Symbols block).
Built by subsetting and merging two Google Noto fonts:

- **Noto Sans Symbols** — zodiac signs (♈–♓) and most planet glyphs
- **Noto Sans Symbols 2** — the Sun glyph (☉, U+2609), absent from Symbols 1

Both are licensed under the SIL Open Font License 1.1
(https://openfontlicense.org). The subset/merge is a permitted derivative;
this font is used only to draw the chart glyphs, which the default Android/iOS
system fonts don't include.
